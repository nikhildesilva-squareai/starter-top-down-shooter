## Overview

In this intermediate-level project, you'll build a **Top-Down Shooter** using Python, Pygame, Particles. Build a top-down shooter with waves of enemies and weapon upgrades. This is a hands-on project where you'll write real code, solve real problems, and build something you can showcase in your portfolio.

This project is part of the **Game Development** course, where you'll build complete games with physics, AI, and player engagement mechanics.

## Approach

Start by setting up Pygame and creating the game window. Build the core game loop (input → update → render). Add game objects one at a time — get each working before adding the next. Add collision detection, scoring, and polish last. Test each feature as you build it.

## Getting Started

```bash

git clone https://github.com/nikhildesilva-squareai/starter-top-down-shooter.git
cd starter-top-down-shooter

# Set up Python environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\\Scripts\\activate
pip install -r requirements.txt

# Run
python main.py
```
